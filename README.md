# Software Development Company Landing Page

Современный лендинг для IT-компании, занимающейся разработкой программного обеспечения. Дизайн создан в Figma и предназначен для презентации услуг компании, проектов и привлечения новых клиентов.

## О проекте

Данный проект представляет собой сайт software development компании. Основная цель лендинга — показать профессиональные услуги в сфере разработки, рассказать о команде и упростить взаимодействие с потенциальными клиентами.

## Секции сайта

### Hero Section
Главный экран с заголовком, кратким описанием компании и кнопками для связи или начала сотрудничества.

### About Company
Информационный блок о компании, опыте работы и направлениях деятельности.

### Services
Секция с услугами компании: веб-разработка, мобильные приложения, UI/UX дизайн, backend-разработка и техническая поддержка.

### Features
Блок с преимуществами компании, включая качество разработки, современные технологии и индивидуальный подход к каждому проекту.

### Portfolio
Секция с примерами выполненных проектов и кейсов компании.

### Team Section
Блок с информацией о команде разработчиков, дизайнеров и специалистов компании.

### Testimonials
Отзывы клиентов о сотрудничестве и качестве выполненных проектов.

### Statistics
Секция со статистикой проектов, клиентов и опыта компании.

### CTA Section
Блок с призывом начать проект или связаться с компанией.

### Footer
Нижняя часть сайта с контактной информацией, ссылками и социальными сетями.

## Особенности дизайна

Макет выполнен в современном минималистичном стиле с удобной структурой, адаптивным интерфейсом и акцентом на профессиональную подачу информации.

## Технологии

Для реализации проекта можно использовать HTML5, CSS3, JavaScript, React.js и Tailwind CSS.

## Responsive Design

Дизайн адаптирован под desktop, tablet и mobile устройства.

## Figma

Проект разработан в Figma с использованием компонентов, Auto Layout и современных принципов UI/UX дизайна.
